# Conheça mais sobre o Mercúrio
O Mercúrio é muito lembrado por ser um metal líquido em temperatura ambiente e também pelo seu papel importante nos termômetros de vidro, mas, na verdade, ele tem participações interessantes ao longo da história da humanidade.
Este site reúne algumas delas.
